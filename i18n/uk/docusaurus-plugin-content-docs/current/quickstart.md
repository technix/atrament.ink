# Швидкий початок

Давайте створимо проєкт на Атраменті **всього за 5 хвилин**.

## Що вам потрібно

- [Node.js](https://nodejs.org/en/download/) версії 22.12 чи вище.
  - Під час встановлення Node.js рекомендується встановити всі залежності.

## Створюємо проєкт

Запустіть цю команду, щоб створити новий проєкт на Атраменті. Ви можете ввести цю команду в Командному Рядку, Powershell, Терміналі, чи вбудованому терміналі вашого редактора коду.

```bash
npx atrament-wizard create my-atrament-game
```

Команда задасть вам декілька питань. Натисніть `Enter`, щоб прийняти налаштування за замовчуванням, ви зможете змінити їх пізніше.

:::tip
В пункті "UI Language" оберіть "Ukrainian". Тоді інтерфейс гри буде українською.
:::

Ця команда також встановить всі необхідні залежності для запуску Атраменту.

## Запускаємо гру

Запустіть сервер:

```bash
cd my-atrament-game
npm run start
```

Команда `cd` змінює поточний каталог. Для роботи з новоствореним проєктом на Атраменті ви маєте перейти в каталог з ним.

Команда `npm run start` збирає веб-інтерфейс Atrament Web UI і робить його доступним за адресою http://localhost:8900/.

## Редагуємо скрипт

Відкрийте файл `root/game/story.ink` і змініть щось в ньому: гра **перезапуститься автоматично** з вашими змінами.

## Публікуємо гру

Щоб опублікувати гру на Атраменті у вигляді HTML-сторінки (подібно до веб-експорту Inky чи до експорту Twine), запустіть цю команду в терміналі:

```bash
npm run build-singlefile
```

Ваша опублікована гра буде знаходитися в каталозі `build/singlefile`.
