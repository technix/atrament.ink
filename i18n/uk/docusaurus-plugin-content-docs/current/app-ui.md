# Ігровий інтерфейс

## Екрани

Застосунок має три основні екрани.

### Стартовий екран

**Стартовий** екран відображає інформацію про гру та головне меню.

![Title screen](/img/doc/app-title.png)

### Ігровий екран

**Ігровий** екран відображає контент гри.

![Game screen](/img/doc/app-game.png)

### Екран "Про гру"

Екран "**про гру**" (налаштовуваний) відображає інформацію про гру.

![About screen](/img/doc/app-about.png)



## Меню застосунку

Натискання кнопки `≡` на екрані викликає меню застосунку. Воно може містити різні розділи в залежності від налаштувань гри та поточного екрану:
- меню завантаження гри (якщо збереження увімкнені)
- меню збереження гри (якщо збереження увімкнені)
- налаштування: повноекранний режим, анімації, гучність звуку, шрифт та його розмір, тема оформлення.

![Меню застосунку](/img/doc/application-menu.png)

## Клавіатурні комбінації

| Клавіша | Опис |
| :--- | :--- |
| 1,2,3... | Вибрати відповідний варіант. |
| Space, Enter | Продовжити історію. |
| Esc | Відкрити/сховати меню застосунку. |
| ~ | (швидко натиснути двічі) Відкрити екран налагодження, якщо воно увімкнено. |
