# Стартовий екран

Стартовий екран гри відображає її назву, ім'я автора та обкладинку.

Він налаштовується за допомогою глобальних тегів:

```
# title: Назва гри
# author: Ім'я автора
# cover: images/game_cover.jpg
```

Ви можете змінити порядок цих елементів або прибрати деякі з них за допомогою глобального тега `#title_screen_layout`:

```
# title_screen_layout: cover, title, author
```

Розмір обкладинки можна змінити, вказавши її ширину другим параметром тега `#cover`:

```
# cover: images/game_cover.jpg 30%
```
