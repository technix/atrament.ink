# Зображення, звуки, відео

## Зображення

### Ілюстрації

Щоб додати ілюстрацію, ви можете використати тег параграфу `#IMAGE`. Ілюстрація буде показана перед текстом параграфу. Розмір ілюстрації визначається автоматично, щоб підлаштуватися під розмір екрану.

```c
=== location_forest
# IMAGE: images/forest.jpg
Ви увійшли до лісу.
```

Також ви можете використати тег розмітки `[picture]`, щоб додати зображення в текст.

```c
=== location_forest
[picture]images/forest.jpg[/picture]
Ви увійшли до лісу.
```

Тег розмітки `[picture]` має такі атрибути:

| Атрибут | Опис                |
| :------ | :------------------------- |
| `width=50%` | Вказує ширину зображення. Висота зображення буде визначена автоматично зі збереженням пропорцій. |
| `leftmargin=0.5em` | Вказує відступ ліворуч від зображення. |
| `rightmargin=0.5em` | Вказує відступ праворуч від зображення. |

```c
[picture width=50% leftmargin=0.5em]images/forest.jpg[/picture]
```

### Зображення в тексті

Щоб додати зображення як частину тексту, використовуйте тег розмітки `[img]`. Розмір зображення буде відповідати розміру шрифта.

```c
Ви знайшли меч: [img]images/sword.jpg[/img]. Він доволі важкий.
```

#### Фон сторінки

Щоб додати фон сторінки на стартовий екран, використовуйте глобальний тег `#background`:

```c
# background: images/forest_bg.jpg
```

Пізніше під час гри ви можете змінити або прибрати його за допомогою тега параграфу `#PAGE_BACKGROUND`:

```c
=== story_knot1
# PAGE_BACKGROUND: images/sea_bg.jpg

=== story_knot2
# PAGE_BACKGROUND: false // прибрати фон сторінки
```

#### Фон сцен

Щоб додати чи прибрати фон для текстових сцен, використовуйте тег параграфу `#BACKGROUND`:

```c
=== story_knot1
# BACKGROUND: images/sea_bg.jpg // додати фон для сцен

=== story_knot2
# BACKGROUND: false // прибрати фон для сцен
```


## Звуки

Звуки відтворюються один раз. Можливо відтворити декілька звуків одночасно.

Щоб відтворити звук, використовуйте тег параграфу `#PLAY_SOUND`. Всі звуки, вказані в параграфі, будуть відтворені одночасно.

```c
=== strangers_visit
# PLAY_SOUND: sounds/doorbell.mp3
Хтось подзвонив у двері.
```

Щоб припинити відтворення звуку, використовуйте тег параграфу `#STOP_SOUND`:

```c
=== doorbell_stops_ringing
# STOP_SOUND: sounds/doorbell.mp3 // зупинити вказаний звук
Дверний двінок замовк.

=== quiet
# STOP_SOUND // зупинити всі звуки
Стало тихо.
```

## Музика

Музика відтворюється зациклено. Можливе відтворення декількох музичних файлів водночас.

Інформація про поточну музику зберігається під час збереження гри. Після завантаження збереженої гри музика починає грати автоматично.

Щоб відтворити музичний файл, використовуйте тег параграфу `#PLAY_MUSIC`. Всі музичні файли, вказані в параграфі, будуть відтворені одночасно.

```c
=== level1
# PLAY_MUSIC: music/level1.mp3
Ви входите в підземелля.
```

Щоб припинити відтворення музичного файлу, використовуйте тег параграфу `#STOP_MUSIC`:

```c
=== exit_dungeon
# STOP_MUSIC: music/level1.mp3 // припинити відтворення вказаного файлу
Ви знову на поверхні.

=== quiet
# STOP_MUSIC // зупинити всю музику
Стало тихо.
```

## Відео

Для відтворення відео, використовуйте тег розмітки `[video]`:

```c
Історія починається...
[video]videos/intro.mp4[/video]
```

Тег розмітки `[video]` має такі атрибути:

| Атрибут | Опис                |
| :------ | :------------------ |
| `loop=false` | Відтворити відео один раз. |
| `muted=true` | Відтворити відео без звуку. |

```c
[video loop=false muted=true]videos/intro.mp4[/video]
```
