# Інтерфейс

Атрамент має декілька налаштованих елементів інтерфейсу:
- панель над текстом
- оверлей
- екран "Про гру" в стартовому меню

## Панель

В верхній панелі відображається назва гри. Щоб створити власну панель, додайте глобальний тег `#toolbar` в головний Ink скрипт, в якому вкажіть функцію, що виводить вміст панелі:

```
# toolbar: player_toolbar
VAR health = 100
VAR strength = 5

=== function player_toolbar()
    Здоров'я: {health} | Сила: {strength}
```

Вміст панелі оновлюється:
- коли гравець робить вибір
- після виклику будь-якої функції Ink

Панель може містити кнопки та інші елементи розмітки.

## Оверлей

Атрамент може відображати довільні дані (інвентар, характеристики персонажа тощо) як оверлей - екран, що перекриває ігровий текст.

Спочатку ви маєте створити функцію, яка виводить вміст оверлею. Потім додайте її до тега розмітки `[button]`. Якщо перший рядок виводу функції - це тег `[title]заголовок[/title]`, цей заголовок буде показаний в панелі.

Приклад панелі та оверлеїв:
```
# toolbar: game_toolbar

=== function game_toolbar()
  [button onclick=inventory]Інвентар[/button]
  [button onclick=stats]Характеристики[/button]

=== function inventory()
  [title]Інвентар[/title]
  У вас є:
  ...

=== function stats()
  [title]Персонаж: {character_name}[/title]
  Здоров'я: {health}
  ...
```

Вміст оверлею може мати кнопки та іншу розмітку. Якщо функція, назначена кнопці, має текстовий вивід, він буде показаний як новий оверлей. Якщо ні, то натискання кнопки призведе до оновлення поточного оверлею.

![Повноекранний оверлей](/img/doc/overlay-fullscreen.png)

Оверлей може відображатися як модальне вікно замість повноекранного. Для цього додайте атрибут `display=modal` до тега кнопки:
```
[button onclick=inventory display=modal]Inventory[/button]
```

![Модальний оверлей](/img/doc/overlay-modal.png)

## Екран "Про гру"

Ви можете додати в гру екран "Про гру" за допомогою глобального тега `#about`. Коли він присутній, в стартовому меню буде пункт "Про гру", який виведе на екран текст, що виводить вказана в тезі `#about` функція:
```
# about: game_about

=== function game_about()
  [banner]Назва гри[/banner]
  Різна інформація про гру
  ...
```
