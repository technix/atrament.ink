# Оповідання

## Зовнішній вигляд

### Розташування сцен

Атрамент виводить зміст сцен, починаючи з центру екрану. Ви можете змінити розташування сцен за допомогою глобального тегу `#scenes_align`, встановивши значення `top` (згори), `center` (по центру) або `bottom` (внизу).

```
# scenes_align: center
``` 

### Поведінка сцен

Атрамент виводить всі текстові параграфи між блоками варіантів вибору як одну сцену. Якщо ви хочете виводити їх рядок за рядком, додайте в ваш сценарій глобальний тег `#continue_maximally` із значенням `false`:

```
# continue_maximally: false
```

## Історія

Атрамент відображає історію попередніх сцен, яку можна гортати. Якщо вам це не потрібно, ви можете вимкнути це за допомогою глобального тегу `#single_scene`:

```
# single_scene
```

Щоб очистити історію сцен вручну, ви можете скористатися тегом параграфу `#CLEAR` - наприклад, коли гравець входить в нову локацію чи починає новий діалог.

```c
=== house_scene
# CLEAR
Ви стоїте посеред поля, перед вами білий будинок.
```

## Перезапуск

Щоб перезапустити гру з самого початку, ви можете скористатися тегом параграфу `#RESTART`:

```c
=== game_over
Гру завершено.
+ [Почати знову]
    #RESTART
    ->DONE
```

## Гіпертекст

В гіпертекстовому режимі варіанти вибору не відображаються під текстом сцени. Ви можете створити посилання із сцени на варіанти вибору.

Щоб увімкнути цей режим для всієї гри, додайте глобальний тег `#hypertext`. Цей режим неможливо вимкнути пізніше.

```c
# hypertext
```

Проте ви можете увімкнути гіпертекстовий режим для певних сцен. Використовуйте тег параграфу `#HYPERTEXT`, щоб увімкнути цей режим для поточної сцени:

```c
=== house_scene
# HYPERTEXT

// посилання використовують текст виборів
Ви стоїте посеред поля, перед вами білий будинок.
[link=Відкрити двері]Двері[/link] будинку зачинені. 
Тут є [link=Подивитися в почтову скриньку]маленька поштова скринька[/link].

+ [Подивитися в почтову скриньку] -> examine_mailbox
+ [Відкрити двері] -> inside_house
```

Якщо вся ваша гра використовує режим гіпертексту, варто додати ці глобальні теги:
* `#single_scene` - вимикає історію сцен
* `#scenes_align: top` - відображає текст сцени згори вікна
