# Архівований ігровий контент

Атрамент підтримує завантаження ігрового контенту з ZIP-файлу, що дає можливість швидко завантажувати асети за рахунок збільшення часу запуску гри. Це має сенс лише для публікації гри як веб-застосунок.

Щоб увімкнути цю функцію, відредагуйте файл `atrament.config.json` і додайте параметр `zip` з бажаним ім'ям архівного файлу:
```
{
  ...
  "game": {
    "path": "game",
    "source": "gamefile.ink",
    "zip": "yourgame.zip"
  }
}
```
:::warning
Цей параметр ігнорується під час розробки або публікації в одиночний файл.
:::
